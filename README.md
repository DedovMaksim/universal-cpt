# Universal CPT

Простой WordPress-плагин для создания Custom Post Types через админку.

## Описание

**Universal CPT** — это лёгкий плагин, который позволяет создавать и регистрировать собственные типы записей (Custom Post Types) без написания кода.

Плагин задуман как минималистичный инструмент:

- без метабоксов
- без таксономий
- без сложных настроек
- без перегрузки интерфейса

Только создание, хранение и регистрация CPT.

## Возможности

- создание Custom Post Types через админку
- редактирование созданных CPT
- хранение конфигураций в `options`
- регистрация CPT через `register_post_type()`
- базовая валидация slug
- поддержка миграций схемы данных

## Что входит в MVP

- список всех созданных CPT
- форма создания CPT
- форма редактирования CPT
- регистрация CPT на `init`
- простая архитектура на основе `options`

## Чего пока нет

- таксономий
- кастомных полей
- метабоксов
- импорта/экспорта JSON
- генерации шаблонов
- расширенных capabilities

##Требования

WordPress 6.x+

PHP 7.4+ (или выше)

##Установка

Скопируйте папку плагина в wp-content/plugins/

Активируйте плагин в админке WordPress

Перейдите в меню Universal CPT

Создайте новый тип записи
